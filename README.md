# Product Hub

A **React.js + Tailwind CSS** project showcasing a searchable product catalog. Users can search for products like clothing, electronics, cosmetics, and see details such as images, descriptions, and prices.

---

## 🚀 Features

- Search bar for filtering products
- Responsive product cards with images, name, and description
- Fast-loading product images (via online URLs)
- Built with **React.js** for UI and **Tailwind CSS** for styling
- Fully functional local project with modular components

---

## 🛠 Technologies Used

- **React.js** – UI library
- **Tailwind CSS** – Utility-first CSS framework
- **JavaScript (ES6)** – Front-end scripting
- **Node.js & npm** – Package management
- **Vite** – Dev server and build tool
